# Phapi Exception
Package contains a set of exceptions matching HTTP error responses. Phapi will take care of logging the error and sending a proper error response when throwing these exceptions.

**Included exceptions:**
* BadGateway
* BadRequest
* Conflict
* Forbidden
* Gone
* InternalServerError
* Locked
* MethodNotAllowed
* NotAcceptable
* NotFound
* NotImplemented
* PaymentRequired
* RequestEntityTooLarge
* RequestTimeout
* ServiceUnavailable
* TooManyRequests
* Unauthorized
* UnprocessableEntity
* UnsupportedMediaType
